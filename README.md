# 🎬 Hệ thống Đặt Vé Xem Phim Trực Tuyến

## Giới thiệu

Dự án được thực hiện trong học phần Công nghệ Phần mềm.

Mục tiêu của hệ thống là hỗ trợ khách hàng:

- Xem thông tin phim
- Xem lịch chiếu
- Chọn ghế
- Thanh toán vé trực tuyến
- Quản lý lịch sử đặt vé
